# PlantPal — Phase Prompts Index

Each file contains the full Claude Code prompt, file list, decisions, and outcome notes for one phase.
Load only the current phase file at the start of a session — do not load all phases at once.

## How to use
1. Check `BUILD_STATUS.md` to find the current phase
2. Open the relevant phase file below
3. Read the **Decisions required** section and answer each question before coding
4. Paste the **Claude Code prompt** when ready to begin
