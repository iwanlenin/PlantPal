Before writing code, config, or scripts — check for best practices relevant to the technology in use and factor them into decisions.

**Why:** User explicitly requested this. Don't just make things work — make them right.

**How to apply:** For each new file or decision, consider: what does the community/official docs recommend? Are there known pitfalls? Apply the best known approach, not just the first working one.
